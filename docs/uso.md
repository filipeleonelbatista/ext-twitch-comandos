# Como usar a extensão – Comandos de áudio (colonogamer)

Esta extensão aparece como um **painel** abaixo do vídeo na live do canal **colonogamer** na Twitch. Com ela você pode enviar comandos de áudio para o chat com um clique (ou toque no celular).

## Onde encontrar

- Na página da live do **colonogamer**, role até a área abaixo do player.
- Procure o painel da extensão de comandos de áudio (nome configurado pelo canal).
- Clique para expandir o painel se estiver recolhido.

## O que você vê

- **Lista de comandos**: botões com os comandos disponíveis para você.
- **Filtro por categoria**: use o filtro (dropdown ou abas) para ver apenas uma categoria, por exemplo:
  - Audios Colono  
  - Audios Diversos  
  - DBD  
  - Audios IF  
  - Ou “Todas” para ver tudo.

## Quem vê o quê

- **Todos (seguidores e visitantes)**  
  - Veem a lista de comandos para **seguidores**.  
  - Podem clicar e enviar esses comandos para o chat.

- **Inscritos (subs)**  
  - Veem a mesma lista de seguidores **e** uma lista extra de comandos de **áudios** (só para inscritos).  
  - Podem clicar em qualquer comando das duas listas.

Se você não for inscrito, pode aparecer uma mensagem incentivando a se inscrever para desbloquear os áudios extras.

## Como enviar um comando

1. Escolha a categoria no filtro (ou deixe em “Todas”).
2. Clique no botão do comando desejado (ou toque, no celular).
3. O comando é enviado para o chat da live em nome da extensão.
4. O áudio correspondente é tocado na live conforme a configuração do canal (bot/OBS etc.).

## Limites e avisos

- A Twitch limita a **12 mensagens por minuto** por viewer. Se você clicar em muitos comandos em pouco tempo, pode aparecer “Aguarde X segundos” ou os botões ficarem temporariamente desabilitados.
- A lista de comandos é atualizada automaticamente a partir das planilhas do canal; se não aparecer um comando novo logo após ser adicionado, aguarde alguns minutos ou recarregue a página.

## Celular

- O painel funciona no app da Twitch (Android/iOS).  
- Use da mesma forma: abra o painel, escolha o filtro e toque no comando para enviar.
